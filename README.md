# Flatironland Ride Selector

A simple Node.js command-line application that allows users to explore different rides at the Flatironland theme park.

## Features
- Interactive terminal prompts using `readline`.
- Validation for specific ride names.
- Dynamic descriptions for selected rides.

## How to Run
1. Ensure you have [Node.js](https://nodejs.org) installed.
2. Clone this repository.
3. Run the program using:
   ```bash
   node index.js
